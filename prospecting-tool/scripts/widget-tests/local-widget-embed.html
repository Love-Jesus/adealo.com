Adealo Widget - Local Development

This page provides embed code for testing your Adealo Widget in a local development environment.

Local Development Only

The embed code on this page is for local development and testing only. It points to your local development server (http://localhost:5173/loader.js) instead of the production server.

For production use, use the standard embed code that points to your Firebase hosting or custom domain.

Local Development Setup

To test the widget locally:

  1. Make sure your local development server is running on port 5173
  2. Copy the loader.js file to your local server's public directory
  3. Ensure your Firebase functions are deployed or emulated locally
  4. Use the embed code above in your test pages

Troubleshooting

  • CORS Issues: If you encounter CORS errors, make sure your Firebase functions allow requests from your local domain
  • 404 Not Found: Ensure the loader.js file is correctly placed in your local server's public directory
  • Function Errors: Check that your Firebase functions are deployed and accessible

Next Steps

After testing locally, deploy your widget to Firebase hosting using the deploy-widget.sh script for more stable testing.
