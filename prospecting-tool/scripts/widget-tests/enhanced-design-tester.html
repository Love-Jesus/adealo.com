Configuration Options:

The enhanced design includes several new configuration options:

  • launcher.useGradient: Toggle between gradient and solid color for the launcher button
  • launcher.pulseAnimation: Enable/disable the pulse animation for the launcher button
  • launcher.shape: Choose between 'circle' and 'rounded-square' for the launcher button
  • launcher.size: Set the size of the launcher button
  • colors.gradient: Configure gradient colors and direction
  • animation.type: Choose from various animation types for the widget
  • chatConfig.typingIndicator: Enable/disable typing indicators
  • chatConfig.showTimestamps: Enable/disable message timestamps
  • content.quickResponses: Configure quick response options
  • content.promotionalContent: Configure promotional content
