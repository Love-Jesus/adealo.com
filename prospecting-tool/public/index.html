Adealo Widget Service

This is the hosting service for the Adealo Widget system. The primary purpose of this service is to deliver the widget loader script and supporting resources to websites that have embedded the Adealo Widget.

Widget Embed Code

To add the Adealo Widget to your website, copy and paste the following code snippet into your HTML:

<!-- Adealo Widget -->
<script>
  (function(w,d,s,o,f,js,fjs){
    w['AdealoWidget']=o;w[o]=w[o]||function(){(w[o].q=w[o].q||[]).push(arguments)};
    w[o].l=1*new Date();js=d.createElement(s);fjs=d.getElementsByTagName(s)[0];
    js.async=1;js.src=f;fjs.parentNode.insertBefore(js,fjs);
  })(window,document,'script','adealo','https://adealo-ce238.web.app/loader.js');
  adealo('init', 'YOUR_WIDGET_ID');
</script>
<!-- End Adealo Widget -->

Replace YOUR_WIDGET_ID with the unique ID of your widget.

Resources

For Developers

This service is part of the Adealo Widget system, which includes:

  • Firebase Hosting (this service)
  • Firebase Functions for widget script generation and interaction tracking
  • Firestore database for widget configuration and analytics

For more information or support, please contact the Adealo team.
