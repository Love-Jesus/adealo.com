Direct Widget URL Test

About This Test

This page demonstrates the direct URL implementation for the widget script, eliminating the need for the redirect function.

The widget is loaded directly from Firebase Hosting using the URL:

https://adealo-ce238.web.app/widget.js

Benefits of Direct URLs:

  • Improved performance by eliminating an HTTP redirect
  • Simplified architecture with fewer components
  • Reduced costs from fewer function invocations
  • Increased reliability by removing a potential point of failure

Implementation

The widget is loaded using a direct script tag:

<script src="https://adealo-ce238.web.app/widget.js"></script>

This replaces the previous implementation that used a Cloud Function redirect:

<script src="https://us-central1-adealo-ce238.cloudfunctions.net/intercomStyleWidgetAdapter"></script>

Test Results

The widget should appear in the bottom-right corner of this page. Click on it to test its functionality.

If the widget loads correctly, it confirms that the direct URL implementation is working as expected.
