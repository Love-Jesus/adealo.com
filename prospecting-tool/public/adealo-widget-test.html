Testing the Adealo Widget with direct Firebase Hosting URLs

About This Page

This page demonstrates the Adealo Widget using the direct Firebase Hosting URL approach. The widget should appear in the bottom right corner of the page.

Note: This page is using the direct URL https://adealo-ce238.web.app/widget.js instead of the Cloud Functions URL.

Widget Features

  • Customizable launcher button (size, shape, color)
  • Multiple screens (home, support, book demo, call me)
  • Light and dark theme support
  • Responsive design
  • Smooth animations
  • Event tracking

Implementation

The widget is implemented with a single line of code:

<script src="https://adealo-ce238.web.app/widget.js" data-widget-id="test-widget-id"></script>

For more advanced customization, you can use the configuration object:

<script>
  window.widgetConfig = {
    id: "test-widget-id",
    design: {
      position: "bottom-right",
      theme: "light",
      colors: {
        primary: "#4a6cf7"
      },
      launcher: {
        size: 60,
        shape: "circle",
        pulseAnimation: true
      }
    },
    content: {
      title: "Chat with Adealo",
      greeting: "Hello ðŸ‘‹",
      tagline: "How can we help you today?"
    }
  };
</script>
